Auto-preview in Firefox

To make auto-preview in Firefox work, follow these steps

  1. Right click on this page and select View Page Info
  2. Click Permissions tab
  3. Uncheck Use default in Install Add-ons
  4. Select Allow

If you don't want to add this site to exception list, you can still preview the colors from preview box.

For Win XP users: to enable custom colors in title bar, install this user style.
